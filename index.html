Install as App

This app can be installed on your device for offline use.

  • Chrome / Edge (Windows/Mac): Click the install icon (⊕) in the address bar, or go to Menu → Install Summer Slam
  • Safari (Mac): File → Add to Dock
  • iPhone/iPad: Share → Add to Home Screen

All data is stored locally on your device and persists between sessions.
